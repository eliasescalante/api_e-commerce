Documentacion de mi API - Elias Escalante - Primera Pre-entrega

Obtener todos los productos

GET /api/products

Obtener el producto por el ID

GET /api/products/:pid

Agregar un producto

POST /api/products

{
    "title": "Product Name",
    "description": "Product Description",
    "code": "Product Code",
    "price": 123.45,
    "status": true,
    "stock": 10,
    "category": "Category",
    "thumbnails": "ruta"
}

actualizar un producto

PUT /api/products/:pid

{
    "title": "Updated Name",
    "description": "Updated Description",
    "price": 150.00
}

Borrar un producto

DELETE /api/products/:pid

Crear un carrito

POST /api/carts

{
    "products": []
}

Obtener el producto del carrito

GET /api/carts/:cid

Agregar un producto al carrito

POST /api/carts/:cid/product/:pid

{
    "quantity": 1
}
